Blinked Controlled Game

For this project I used both python and C# to create a game which uses the users’ binks to control the y direction of the player. I used to python to do the visual processing of the blinks as C# was not well suited to the task. Due to this I had to use the TCP/IP protocol to send information from the separate scripts using local ports. I then used Unity to handle the graphical side of the project. The game was a simple galactica style game. The link to the games' GitHub is below;
